**AI-Powered Stock Trading Assistant & Sentiment Analyzer**

An automated **n8n workflow** that combines multi-timeframe technical analysis with AI-driven news sentiment to generate daily trading signals (Buy/Sell/Hold). This tool leverages Google Gemini models to synthesize market data and news into actionable insights, delivered directly to your email.

---

## üöÄ Key Features

* **Automated Market Analysis:** Fetches real-time technical data for 1-minute, 15-minute, and 1-hour candles via the **Twelve Data API**.
* **AI Sentiment Engine:** Scrapes the latest relevant news using **NewsAPI** and analyzes market sentiment (Positive/Neutral/Negative) using **Google Gemini**.
* **Intelligent Trading Agent:** Uses a LangChain AI Agent to act as an "Expert Day Trader," combining technical indicators and sentiment scores to produce a unified recommendation with Entry, Stop-Loss, and Target prices.
* **Performance Tracking:** Automatically logs every signal and summary into **Google Sheets** for historical analysis.
* **Daily Briefing:** Formats the analysis into a clean HTML newsletter and delivers it via **Gmail** every morning.
* **Multi-Asset Support:** Can be configured to analyze any list of symbols (Stocks, Crypto, Forex) supported by the data provider.

## ‚öôÔ∏è Workflow Architecture

1.  **Trigger:** Runs automatically on a daily schedule (e.g., 6:00 AM).
2.  **Input:** Reads a configurable list of asset symbols (defined in a Code Node).
3.  **Data Fetching:** Parallel execution to retrieve technical candles (1m, 15m, 1h) and recent news articles.
4.  **LLM Processing:** * *Step 1:* Analyzes news headlines to determine a sentiment score (-1 to 1).
    * *Step 2:* Passes technical data + sentiment to the AI Agent to generate the trade setup.
5.  **Output:** * Appends data to a Google Sheet.
    * Compiles a summary email using HTML styling.
    * Sends the briefing via Gmail.

## üì¶ Setup & Installation

1.  **Import Workflow:**
    * Download the `.json` file from this repository.
    * In your n8n dashboard, click **Add Workflow** > **Import from file**.
2.  **Configure Credentials:**
    * Update the credentials for Google Sheets, Gmail, and Gemini in the respective nodes.
    * Enter your API keys for Twelve Data and NewsAPI in the HTTP Request nodes.
3.  **Define Assets:**
    * Open the `Code` node at the beginning of the workflow.
    * Edit the JSON array to include the symbols you want to track (e.g., `return [ { "symbol": "YOUR_ASSET" }, ... ];`).
4.  **Setup Google Sheet:**
    * Create a new Google Sheet.
    * Add headers (e.g., `Symbol`, `Summary`, `Date`).
    * Update the Sheet ID in the Google Sheets nodes.
5.  **Activate:** Toggle the workflow to **Active**.
